Move Language AI Assistant - Project Summary
What It Is
A TypeScript-based AI assistant that helps developers write Move smart contracts (the language used by Sui and Aptos blockchains).
Key Features

Code Generation

Generates complete Move smart contracts
Token/coin contracts with transfers
NFT collections with minting
Marketplace contracts
Custom modules


Concept Explanation

Explains Move-specific concepts (abilities, resources, signer, acquires, etc.)
Helps developers learn Move language features


Code Review

Analyzes Move code for common mistakes
Checks for missing acquires declarations
Validates struct abilities
Suggests best practices



Technical Stack

Language: TypeScript (no Python, Java, or JavaScript)
Interface: Interactive CLI menu + React web playground
Runtime: Node.js with ts-node

How It Works
The AI uses pattern matching and templates to:

Understand user intent ("create a token")
Generate appropriate Move code with custom parameters
Provide educational explanations
Review code for issue
